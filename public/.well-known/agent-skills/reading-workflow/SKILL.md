# Reading Workflow

Use this skill when users want concise summaries of site content.

## Workflow

1. Start from `/posts/` or `/notes/` indexes.
2. Open the most relevant entry pages.
3. Produce a short summary with the source URL.
4. Suggest related posts if available.

## Output

- Keep summaries brief and factual.
- Include canonical page links for follow-up reading.
