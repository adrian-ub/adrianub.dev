# Site Navigation

Use this skill to quickly route users to the most relevant section.

## Sections

- `/posts/` for blog articles.
- `/notes/` for shorter technical notes.
- `/projects/` for portfolio projects.
- `/sponsors-list/` for sponsorship information.

## Guidance

1. Prefer direct links to the most specific page when possible.
2. If the user intent is unclear, suggest the matching section and ask for confirmation.
